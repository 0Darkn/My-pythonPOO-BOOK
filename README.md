lista completa dos principais conceitos de Programação Orientada a Objetos inclui:
 * Classes: Moldes para criar que definem atributos (características) e métodos (comportamentos).
 * Objetos: Instâncias concretas das classes, com os seus próprios atributos e métodos.
 * Herança: Permite que novas classes (classes filhas) herdem atributos e métodos de classes existentes (classes pai), promovendo a reutilização de código.
 * Encapsulamento: Consiste em esconder os detalhes de implementação de um objeto, controlando o acesso externo a ele.
 * Polimorfismo: Permite que objetos de classes diferentes respondam da mesma forma a uma mensagem, adaptando a sua implementação.
 * Abstração: Foca-se na representação das características essenciais de um objeto, ignorando os detalhes não relevantes.
